## Detailed Voice-to-Action Process

```
Spoken Command: "Please pick up the red block from the table"

Step 1: Voice Processing
┌─────────────────────────────────────────────────────────────┐
│ Audio Input → Noise Reduction → Speech Recognition → Text  │
│ "Please pick up the red block from the table"              │
└─────────────────────────────────────────────────────────────┘

Step 2: Language Understanding
┌─────────────────────────────────────────────────────────────┐
│ Text → Tokenization → Intent Recognition → Entity Extraction│
│ Intent: GRASP_OBJECT                                        │
│ Entities: {object: "red block", location: "table"}         │
└─────────────────────────────────────────────────────────────┘

Step 3: Task Planning
┌─────────────────────────────────────────────────────────────┐
│ Intent + Entities → Task Decomposition → Action Sequence   │
│ 1. NAVIGATE(location="table")                              │
│ 2. PERCEIVE(object="red block")                            │
│ 3. PLAN_GRASP(object="red block")                          │
│ 4. EXECUTE_GRASP(object="red block")                       │
└─────────────────────────────────────────────────────────────┘

Step 4: Action Execution
┌─────────────────────────────────────────────────────────────┐
│ Action Sequence → ROS 2 Commands → Robot Execution         │
│ Execute each action with feedback and error handling       │
└─────────────────────────────────────────────────────────────┘

Step 5: Perception Feedback
┌─────────────────────────────────────────────────────────────┐
│ Environment → Sensors → Processing → Updated State         │
│ Confirm success / Handle errors / Update world model       │
└─────────────────────────────────────────────────────────────┘
```
