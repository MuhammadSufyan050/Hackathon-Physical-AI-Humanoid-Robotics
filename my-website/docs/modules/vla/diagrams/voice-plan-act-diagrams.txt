```
┌─────────────────┐    ┌──────────────────┐    ┌──────────────────┐
│   Voice Input   │    │  Cognitive       │    │    Action        │
│   Processing    │───▶│  Planning        │───▶│  Execution       │
│                 │    │                  │    │                  │
│  - Speech Rec.  │    │  - Intent Extr.  │    │  - Motion Plan.  │
│  - ASR System   │    │  - Task Plan.    │    │  - Control Sys.  │
│  - Transcr.     │    │  - Context A.    │    │  - Robot Driver  │
└─────────────────┘    └──────────────────┘    └──────────────────┘
         │                       │                       │
         ▼                       ▼                       ▼
┌─────────────────┐    ┌──────────────────┐    ┌──────────────────┐
│  Audio Signal   │    │  Structured      │    │  Executable      │
│                 │    │  Command         │    │  Actions         │
│  "Pick up the  │    │  Intent: GRASP   │    │  1. NAVIGATE()   │
│  red block"     │    │  Object: red    │    │  2. PERCEIVE()   │
│                 │    │  block          │    │  3. GRASP()      │
└─────────────────┘    │  Location: here │    │  4. CONFIRM()    │
                       └──────────────────┘    └──────────────────┘
```
